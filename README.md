# Dyad Custom (xkamara0 Edition)

Dyad est un constructeur d'applications IA local, open-source et entièrement sous votre contrôle. Cette version est une copie personnalisée maintenue par **xkamara0**.

Basé sur le projet original : [https://github.com/dyad-sh/dyad](https://github.com/dyad-sh/dyad)

## Caractéristiques

*   **Local** : Rapide, privé et sans verrouillage.
*   **Vos propres clés** : Utilisez vos propres clés API IA.
*   **Multiplateforme** : Fonctionne sur Mac et Windows.

## Installation

```bash
npm install
npm start
```
